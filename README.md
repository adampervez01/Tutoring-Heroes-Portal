# Tutoring Heroes Portal

A comprehensive tutor dashboard designed to help educators manage lessons, track student progress, and access essential resources—all in one organized place.

## Features

- 📅 **Today's Lessons** - View scheduled lessons with times, subjects, and learning focus areas
- 📝 **Parent Feedback Tracker** - Track feedback submission deadlines with color-coded priority status
- 👥 **My Students** - Quick student snapshots including subjects, levels, and learning focus
- 🗂️ **Resource Library** - Organized teaching resources by subject and exam pathway
- 📘 **Tutor Handbook** - Essential guidance on best practices and support protocols
- 🔗 **Quick Links** - One-click access to forms, timesheets, and support channels
- 🔔 **Admin Reminders** - Smart prompts for timesheets, feedback, and meetings

## Tech Stack

- **React 18** - UI library
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **JavaScript (ES6+)**

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/adampervez01/Tutoring-Heroes-Portal.git
cd Tutoring-Heroes-Portal
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```
